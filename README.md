# Sparx-Maths-TimesTables
This means that you just need to click s on your keyboard and it will peform the calculation. Currently it can only do multiplication.

**Installation**

Needed modules

Pypiwin32
```bash
pip install pypiwin32
```
Mouse
```bash
pip install mouse
```
Easyocr
```bash
pip install easyocr
```
Pyautogui
```bash
pip install pyautogui
```
Pynput
```bash
pip install pynput
```
